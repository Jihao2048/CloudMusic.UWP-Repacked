## 简介

## 安装

由于 Windows 只允许安装来自**微软商店**或者**企业开发者**的 appx/msix 软件包，<br />
所以我们需要先安装**自签名证书**来伪装成企业开发者，才能安装第三方软件包。

## 方法

1. 右键或长按选择以管理员身份打开 `一键安装根证书.bat` ，根据提示操作即可。
2. 当显示内容包含 `证书 "MyCA Root" 添加到存储` 或者 `证书 "MyCA Root" 已经在存储中` 说明安装成功。
3. 下载合适的 `appx` 软件包，在开始菜单中找到powershell，以管理员身份打开，输入 `Add-AppxPackage <路径>` 即可安装，`Add-AppxPackage` 不严格要求大小写。
4. 打开软件。如有问题，请查阅下方的 [已知问题及解决方法](#已知问题及解决方法) 。

> [!NOTE]
> 某些杀毒软件可能会将证书安装工具误报为病毒，<br />
> 您可能需要暂时关闭杀毒软件或者添加白名单才能使用。<br />

## 已知问题及解决方法

## 提示网络错误无法登录的解决方案

![演示动图](setprotocol.gif)

1. 点击 左下角“未登录”头像 → 关于网易云音乐，来到关于界面
2. 对界面右上角的网易云音乐 Logo ，鼠标左键连击五次，然后右键单击，会弹出调试对话框，如果不行就多试几次。如是触屏设备，请外接鼠标。
3. 将服务地址 `http://music.163.com` 中的 `http` 改为 `https`，即改为 `https://music.163.com`，然后确认
4. 手动重启应用后即可使用手机号及邮箱正常登录

&nbsp;

## 提示存在风险无法登录的解决方案

**请尝试以下方法：**

1. 如果您使用了任何抓包/网络过滤/加速器/代理软件，请将它们关闭后再试
2. 按照上述步骤改为HTTPS后，等待24小时后使用 QQ 扫码登录。

## 提示“不可信设备”提示的解决方案

1. 在手机端登录账号
2. 进入设置关闭登录保护
3. 正常登录 UWP 客户端
4. 如有必要，在 UWP 登录成功后可重新打开保护。

#### 2. 下载时文件存储失败
 在 22H2 及更高版本的 Windows11 中下载音乐时提示文件存储失败。
 
1. 下载 [路径授权工具](../../../raw/main/tools/授权访问外部路径.exe)
2. 在Windows defender 中关闭“实时保护”。如果使用其他杀毒软件，也需要关闭。
3. 右键或长按选择“以管理员身份运行” `授权访问外部路径.exe` ，根据提示操作即可。

#### 3. 安装appx时“无法安装相关项”
1. 打开 dependencies 文件夹，根据你的系统架构选择对应的文件夹。
2. 右击打开文件夹，在资源管理器路径输入框里输入powershell，在里面挨个输入 `Add-AppxPackage <文件名称>` 安装依赖包。`Add-AppxPackage` 不严格要求大小写。

#### 4. 我是黑胶会员，但播放/下载时显示需要开通
1. 手动退出登录，然后重新登录账号，即可恢复正常。<br>
2. 如果仍然无效，就需要去看看您的黑胶会员是不是真的过期了。<br>

#### 4. Windows on ARM 更新 24H2+ 后无法安装和使用
这个平台太罕见了吧!但是我觉得，搭载这个平台和系统的电脑，应该都是非常新的，所以就不建议你去用这么十年前出的小玩意了，可以换官方的客户端，或者更新的第三方客户端。

#### 5. 不支持HiRes和超清母带等新音质
UWP客户端推出时还没这些~~提升用户体验的~~音质级别，停止更新后自然没有跟进。<br>
如果你不是那种特别猎奇的老设备，比如Baytrail上网本，或者说Windows Phone，那就可以去用官方的客户端，或者第三方客户端。
